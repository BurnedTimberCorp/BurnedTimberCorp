# Images Folder

This folder contains all product images and gallery photos for the Burned Timber Corp website.

## Subfolder Structure

### `/products`
Product showcase images:
- `baskets/` - Engraved wooden basket images
- `plaques/` - Engraved plaque images
- `ornaments/` - Engraved ornament and decoration images

### `/gallery`
Portfolio and gallery images:
- Before/after shots
- Customer projects
- Design inspiration

### `/hero`
Hero section and banner images:
- Main website hero background
- Feature banners

## How to Add Images

1. Create subfolders as needed (e.g., `baskets/`, `plaques/`, `ornaments/`)
2. Add your image files (.jpg, .png, .webp)
3. Use descriptive filenames (e.g., `engraved-basket-001.jpg`)
4. Update the `index.html` file to reference your images

## Image Recommendations

- **Optimal Format:** JPG (quality), PNG (transparency), WebP (best compression)
- **Recommended Size:** 800x600px minimum for product images
- **File Size:** Keep under 500KB for faster loading
- **Resolution:** High-res (300dpi) for print-quality appearance
